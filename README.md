# Physics Puzzler

Small browser-based 2D physics puzzle game using [Matter.js]. Works on macOS (and any OS with a modern browser). Great for hosting on GitHub Pages.

## Features
- 3 example levels (push, button-door, stack+ramp)
- Drag objects with mouse / touch
- Simple level system, easy to add more levels
- Single-file JS (easy to edit)

## How to run locally (mac)
1. Clone or download the repo.
2. Open `index.html` in your browser, **or** serve via a local HTTP server:
   ```bash
   # from repo root
   python3 -m http.server 8000
   # then open http://localhost:8000
